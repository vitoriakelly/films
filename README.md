# Films Explorer

Aplicacao web para busca de filmes e series utilizando **React** e **Tailwind CSS**, com dados em tempo real da API da **TVMaze** (gratuita e sem chave).

## Funcionalidades

- Busca de filmes por titulo com atualizacao inteligente (debounce)
- Secao de filmes em alta da semana
- Secao de proximos lancamentos
- Pagina de detalhes com:
  - sinopse
  - generos
  - nota
  - duracao
  - elenco principal
  - trailer (YouTube), quando disponivel
- Interface moderna com foco em experiencia intuitiva

## Tecnologias

- React + Vite
- React Router
- Tailwind CSS
- Lucide React (icones)
- API TVMaze (sem chave)

## Como rodar

1. Instale dependencias:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm run dev
```

## Objetivo do projeto

Este projeto foi criado para praticar e evoluir habilidades de front-end, com foco em:

- Consumo de API REST
- Componentizacao com React
- Gerenciamento de estado e side-effects
- Construcao de interfaces responsivas e modernas
